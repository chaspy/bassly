# 分数コード(スラッシュコード) — ベーシストへの直接指示

「F#/Bb」のような表記は「F#のコードを、ベースはBbで支えて」という意味。
左がコード、右(スラッシュの後)がベースの音。バンド譜の中で唯一、
ベーシストに名指しで来る指示なので、見落とすと存在感が出ない。

なぜ使われるか:
- ベースラインを滑らかにするため。コードはF#のままでも、
  ベースだけ3度(Bb)に降りると、次のコードへ階段がつながる
- 例: B → F#/Bb → Ebm と進むと、ベースは B → Bb → Eb と
  半音+αで滑らかに降りていける(コードを聞くと壮大、ベースは省エネ)

弾き方のコツ:
- 指定された音(スラッシュの右側)がその小節の「R扱い」。堂々と伸ばす
- 指定音はたいてい元コードの3度か5度なので、外れる心配はない

この曲では: bar 9・14・26などのF#/Bb(=Gb/Bb)が頻出。
BbはF#の3度なので、Bbmから続けて弾くと「同じBbのまま景色が変わる」
面白い瞬間になっている。
