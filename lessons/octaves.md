# オクターブ — 同じ音の上下で立体感を作る

オクターブは「同じ名前の音の1つ上(下)」。周波数がちょうど2倍なので、
違う高さなのに同じ音に聞こえる。ベースラインに高低の立体感を作る
最も安全な方法(音を外すリスクがゼロ)。

指板では必ず同じ形:
- 「2弦上・2フレット右」が1オクターブ上。どこでも、どのキーでも同じ
- 例: E弦2フレットの音の1オクターブ上は、D弦4フレット

使いどころ:
- ルート連打が単調になってきた時、何発かをオクターブ上に置き換える
- 低いオクターブ(5弦のLow B側)に落とすと一気に重心が下がる。
  サビ前の「溜め」や曲の最低点で効く

この曲では: bar 9のBbペダルは低いF#と高いF#でBbを挟む形。
bar 27・81・113ではLow B弦の低いC#に沈む(5弦の見せ場)。
bar 35はEbを高低オクターブで行き来してサビの山を支える。
