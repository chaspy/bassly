# ディグリー — コード進行を数字で覚える

度数の考え方はコード進行にも使える。キーの主音を1として、
コードのルートがスケールの何番目かを数字で呼ぶ。これがディグリー。

  キーがF#メジャーなら: F#=1  G#m=2  A#m=3  B=4  C#=5  E♭m=6  (7=E#dim)

なぜ数字で覚えるのか:
- コード名(文字)の暗記はキーが変わると全部無効。数字はどのキーでも同じ
- 有名な進行は数字で名前がついている。王道進行=4561、
  丸の内サディスティック(丸サ進行)=4536、カノン進行=15634145
- 「この曲のサビは4→5→1→6」と覚えれば、次の曲で同じ数字列に
  出会ったとき「あ、アレだ」になる。進行が語彙になる

メジャーとマイナーは覚えなくていい:
- キーの中(ダイアトニック)なら質は自動で決まる(1,4,5=メジャー、2,3,6=マイナー)
- だから数字だけで十分。例外(キー外のコード)だけ♭付きなどで目立つ

この曲では:
- イントロリフ: 1→4→5→6 (主音から始めて王道の並びへ)
- サビ: 4→5→1→6 — 王道進行4561の並べ替え。日本のポップスの心臓部
- Aメロ: 4→1→5→6 の変奏。ベースは先乗りで次の数字へ滑り込む
- チャートの水色の数字がこれ。同じ数字列を探すと、この曲が
  少数の進行の使い回しでできていることが見える
