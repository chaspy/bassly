# 4度チューニング — 「形の使い回し」が成立する理由

ベースの隣り合う弦は、どのペアも完全4度(5半音)間隔で調弦されている。

  B → E → A → D → G  (どの間も +4度)

この「全ペア均等」こそが、度数=指の形をどこでも使い回せる根拠。
弦間の音程差が場所によって違ったら、形は場所ごとに補正が必要になる。

均等だから成立する形(どの弦ペア・どのフレットでも同じ):
- 同フレット・1弦下 = 下の5度 (移動ゼロ。ベースラインに一番よく出る)
- 1弦上・2フレット右 = 5度(上)
- 1弦上・同フレット = 4度
- 2弦上・2フレット右 = オクターブ
- 2弦上・同フレット = ♭7

「隣の弦 = 5フレットのシフト」:
- 4度 = 5半音なので、隣の弦は同じ音の並びがちょうど5フレットずれたもの
- B弦7f = E弦2f。B弦の指板が分からなくなったら「E弦の5フレット右」と読み替える

ギターとの対比(豆知識):
- ギター(EADGBE)はG弦とB弦の間だけ長3度で、形がそこで1フレット崩れる
- ベースは例外なしの全4度。理論を指の形に直結できる楽器として
  ギターより素直にできている(6弦ベースも全4度)

対する低いG#)が「同フレット・1弦下」の実例。
